# Agenda Sorties — Montpellier · Nîmes · Sète · Béziers

Page personnelle de sorties culturelles et concerts pour l'été 2026.

**URL publique :** https://lepachapacha.github.io/agenda-montpellier

## Ajouter un événement

Éditer uniquement `agenda-config.md` :

```
- DATE | TITRE | CATEGORIE | LIEU | NOTE | AVEC_FILS(oui/non) | ETOILES(1-3)
```

Catégories : `festival` `concert` `theatre` `expo` `danse` `jazz` `classique` `electro` `feria` `cinema` `activite`
